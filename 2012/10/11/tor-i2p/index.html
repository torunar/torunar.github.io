# /usr/bin/torunar

Tor и I2P в Linux незаметно от санитаров

2012/10/11

Для тех, кто не хочет тратить драгоценные 15 минут в Google.

Tor

  1. Установка Tor

    sudo apt-get install tor

  2. Запуск Tor

    sudo service tor start

I2P

  1. Скачивание и установка i2prouter

  2. Запуск роутера:

    i2prouter start

  3. Добавление подписок в адресную книгу:

Настройка браузера (на примере Firefox)

  1. Установка Proxy Selector

  2. Настройка прокси для Tor:

    Тип прокси: Socks v5
    Host: localhost
    Port: 9050

  3. Настройка прокси для I2P:

    Тип прокси: HTTP
    Host: localhost
    Port: 4444

  4. Настройка DNS для использования .onion-сервисов:

    На странице расширенных настроек нужно выставить настройку network.proxy.socks_remote_dns в true
