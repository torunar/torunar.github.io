# /usr/bin/torunar

Cider: теперь с RSS и поддержкой Disqus

2018/08/20

Маленький, но гордый блоговый движок обрастает фишками.

На выходных допилил для Cider две приятные плюшки: генерацию RSS лент и возможность оставлять комментарии через Disqus (по просьбе @u_frds).

RSS-лента содержит посты с главной — добавляйте в свои читалки новостей, чтобы не пропустить редкие обновления.

Комментарии доступны для всех постов.
